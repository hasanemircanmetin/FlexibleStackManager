# FlexibleStackManager
 StackManager <br>
 Obj = will be stacked object
 Stack Parent Obj = stacked objects will the child of this object and positions will be assigned relative to its position<br>
 Space Between Objects = self-explanatory<br>
 Max Stacked Object Count = stack limit in x, y, z<br>
 
 StackObj() = stacks the object in an available position<br>
 ReturnToObjPool() = return obj to pool when you are done with it!<br>
